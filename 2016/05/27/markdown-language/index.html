Markdown基本语法说明

发表于 2016-05-27   |   分类于 实用工具   |  

标题

  • 方式一：通过在文字下方添加“=”和“-”，===== 表示高阶标题，——— 表示次阶标题。
  • 方式二：在文字开头加上 “#”，通过“#”数量表示几级标题。（一共只有1~6级标题，1级标题字体最大）

目录

  • 使用 [TOC] 生成目录。如一开始的目录所示。

列表

  • 无序列表：在文字开头添加(, +, and -)实现无序列表。但是要注意在(, +, and -)和文字之间需要添加空格。（建议：一个文档中只是用一种无序列表的表示方式）
  • 有序列表：使用数字后面跟上句号。比如1. 2. 3.

列表嵌套

  • 通过列表标志前面加4个空格或tab键缩进完成。

引用和注释

  • 在需要引用的文字前面加上”>”,比如 “> 引用文字 “，（当>和文字之间添加五个blank时，块注释的文字会有变化。）
  • “>>” 表示引用里面再套一层引用，依次类推

斜体

  • 将需要设置为斜体的文字两端使用1个“*”或者“_”夹起来。

粗体

  • 将需要设置为斜体的文字两端使用2个“*”或者“_”夹起来

代码块

  • 使用```表示代码块,比如如下代码：

```bash
while [ $k -eq 1 ] ; do
　　echo “true”
done
```
显示效果如下：

  • 支持的语言：actionscript, apache, bash, clojure, cmake, coffeescript, cpp, cs, css, d, delphi, django, erlang, go, haskell, html, http, ini, java, javascript, json, lisp, lua, markdown, matlab, nginx, objectivec, perl, php, python, r, ruby, scala, smalltalk, sql, tex, vbscript, xml
  • 也可以使用 4 空格缩进，再贴上代码，实现相同的的效果
  • 非常短的代码片段也即是行内代码，只想高亮语句中的某个函数名或关键字，可以用`blockquote`。（`不是单引号而是左上角的ESC下面~中的`）

空格

  • 这是一个技巧，非语法，可以切换到输入法全角字符，然后打空格实现空格效果，markdown语法本身未提供空格
