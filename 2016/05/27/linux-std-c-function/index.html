Linux 标准C语言函数总结

发表于 2016-05-27   |   分类于 C语言   |  

文件操作函数：

1. fopen：打开文件：FILE* fopen(const char *name, const char *mod);  
    返回值：成功：FILE* 失败：NULL
        其中mod的取值有：
            r 打开只读文件，该文件必须存在。
            r+ 打开可读写的文件，该文件必须存在。
            w 打开只写文件，若文件存在则文件长度清为0，即该文件内容会消失。若文件不存在则建立该文件。
            w+ 打开可读写文件，若文件存在则文件长度清为零，即该文件内容会消失。若文件不存在则建立该文件。
            a 以附加的方式打开只写文件。若文件不存在，则会建立该文件，如果文件存在，写入的数据会被加到文件尾，即文件原先的内容会被保留。（EOF符保留）
            a+ 以附加方式打开可读写的文件。若文件不存在，则会建立该文件，如果文件存在，写入的数据会被加到文件尾后，即文件原先的内容会被保留。 （原来的EOF符不保留）
            还可加入“b”表示二进制文件操作。
            由此我们使用时，经常使用方法有，
              r+ 读写，必须存在，文件指针在开始？ 
              w+ 清空读写，不存在则创建
              a+ 追加读写，不存在则创建，文件指针在结束
2. fflush：清空缓冲区写入。int fflush(FILE *stream);  成功：0，失败：EOF
3. fclose:关闭文件流。int fclose(FILE *stream);              成功：0，失败：EOF
4. remove：删除文件。 int remove(char *name)； 成功：0
5. rename:重命名文件。int rename(char *oldname, char *newname);  成功：0
## 格式化输入输出函数： ## 
1. int printf(const char *format, ...);
2. int sprintf(char *s, char *format,……)；打印到字符串s。
        返回实际写入字节数。出错返回负数
        常用场合：.打印整数到字符串 sprintf(s, "%d", a);
              .打印十进制为十六进制 sprintf(s, "%x", 1234);
              .控制打印浮点数格式 sprintf(s, "%.3f", 1234.3456);
              .连接多个字符串 sprintf(s, "%.7s%.5s", "1234","sdfgh");在连接两个字符数组，非字符串时通过制定精度可完成strcat不能完成的工作。 
                            还可以动态制定精度，sprintf(s, "%.*s%.*s", a1, "1234", a2, "sdfgh"),其中a1，a2为动态精度变量，表示最长取字符串中多少长度
              .打印地址 sprintf(s, "%08u", a);
3. int fprintf(FILE *stream, const char *format, ...); 按照format格式写入到文件流中。
4. int scanf(const char *format, ...); 标准输入输入
5. int fscanf(FILE *stream, const char *format, ...); 文件输入
6. int sscanf(const char * s, const char *format, ...); 字符串输入
## 字符输入/输出函数： ## 
1. int fgetc(FILE *stream); 从文件中读出一字符。到文件末尾或出错返回EOF。
2. int fputc(int c, FILE *stream); 将字符c写入FILE中。出错返回EOF。
3. int getchar();标准输入读入一个字符。以Enter结束输入，最后的回车符也会放入缓冲区。
4. int putchar(int c);输出到标准输出。
5. int ungetc(int c, FILE *stream); 回写到文件 出错返回EOF。
6. char *fgets(char *s, int n, FILE *stream); 

        读出最多n-1个字符，或遇到换行符终止存入s，并在*s上后加'\0', 遇到换行符会将换行符写入字符串，并不舍弃。
        到达文件末尾或出错，返回NULL，可用feof，或ferror判断是哪一种
7. int fputs(const char *s, FILE *stream);        
        向指定的文件写入一个字符串（不自动写入'\0'）。成功写入一个字符串后，文件的位置指针会自动后移。
        成功返回0；否则返回EOR
8. char *gets(char *s); 
        从标准输入读入一个输入行到s，并把末尾换行符置换为'\0'，也就是会舍弃换行符。注意s不要溢出。
        可用feof，或ferror判断是哪一种
9. size_t fread(void* ptr, size_t size, size_t nobj, FILE *stream);
        从stream中最多读取size个长度为nobj的对象到ptr中。
        返回读取的对象个数。若出错需要通过feof或ferror判断原因。
10. size_t fwrite(const void* ptr, size_t size, size_t nobj, FILE *stream);
        将ptr中size*nobj个字节的数据写入文件流。
        返回写入对象个数。若出错需要通过feof或ferror判断原因。

字符测试函数：ctype.h

若为真返回非0值，若为假返回0值。
1. int isalnum(int c) 是否字母(包括大小写)或数字
2. int isalpha(int c) 是否字母(包括大小写)
3. int isdigit(int c) 是否十进制数字
4. int islower(int c) 是否小写字母
5. int isupper(int c) 是否大写字母

大小写转换：

1. int tolower(int c) 转小写，若c不满足大写，返回c本身
2. int toupper(int c) 转大写，若c不满足小写，返回c本身

字符串函数：string.h

下面函数中，变量d，s类型为char *，cd和cs型为const char *， n类型为size_t, c类型为int。一般d标识为目的地或返回地，后面s为源
1. char *strcpy(d,cs)  
        将cs(包括'\0')复制到d，返回d。 d和cs不可重叠，d要足够大。
2. char *strncpy(d,cs,n) 
        最多将cs的n个字符复制到d中。
        若cs前n个字节不以NULL结尾,则d末尾不会自动加'\0'，需手动加。
        若cs长度小于n个，则补NULL到d直到n个。                                                        
3. char *strcat(d,cs)
        把cs所指字符串的前n个字符添加到d结尾处(覆盖d结尾处的'\0')并添加'\0'.
4. char *strncat(d,cs,n)
        把cs所指字符串的前n个字符添加到d结尾处(覆盖d结尾处的'\0')并添加'\0'.
        若cs前n个字符不以NULL结尾，也会自动在d末尾加NULL
5. int strcmp(cd,cs)
        比较cd，cs字符串大小。
        当cdcs时，返回值>0
        比如"aaa" < "aaab"
6. int strncmp(cd,cs,n)
        比较cd和cs的前n个字符。
7. char *strchr(cd,c)
        返回字符c在cd中第一次出现位置的指针，若cd不包含c，则返回NULL
8. char *strrchr(cd,c)
        返回字符c在cd中最后一次出现位置的指针，若cd不包含c，则返回NULL
        若仅出现一次，则strchr和strrchr返回同样的非NULL位置。
9. char *strstr(cd,cs)
        返回字符串cs在cd中第一次匹配的位置，若cd不包含cs，则返回NULL
        依此可以判断cd是否包含cs，以及递归判断从而判断cd包含cs的个数。
10. strlen(cd)
        字符串长度
11. size_t strspn(cd,cs)
        返回字符串cd开头连续包含字符串cs内的字符数目。
        比如strspn("Hello World","klleHo");返回5
12. size_t strcspn(cd,cs)
        返回字符串cd开头连续不包含字符串cs内的字符数目。
        比如strspn("Hello World","kl");返回2

下面函数中对应上面的char *变为void *类型，mem函数直接内存操作，与字符串函数相比，不会考虑'\0'的影响。
13. void *memcpy(d,cs,n)
        将cs中n字符拷贝到d中，与strncpy相比，一定会拷贝n字符。
14. void *memmove(d,cs,n)
        将cs中n字符拷贝到d中，与memcpy相比，d和cs内存重叠时仍能工作正常，但增加了开销。
        原理为：当源内存的首地址等于目标内存的首地址时，不进行任何拷贝
                当源内存的首地址大于目标内存的首地址时，实行正向拷贝
                当源内存的首地址小于目标内存的首地址时，实行反向拷贝
15. void *memcmp(cd,cs,n) 
        与strncmp类似，但memcmp一定会比较n个字符，而不受'\0'影响。
16. void *memchr(cd,c,n)
        找到c在cd的前n个字符中的第一次匹配，若未找到，返回NULL
17. void *mem(d,c,n)
        将d中的前n个字符用c替换，常用于初始化，比如memset(array,0,sizeof(array));

实用函数:stdlib.h

1. void *malloc(size_t size)
        分配size个字节的内存，注意分配后最好显式做类型转换
2. void free(void *p)
        回收内存
3. double atof(const char *s)
        将字符串转化为double类型。
        atof()会扫描参数s字符串，跳过前面的空格字符，直到遇上数字或正负符号才开始做转换，而再遇到非数字或字符串结束时('\0')才结束转换，并将结果返回。
        参数s字符串可包含正负号、小数点或E(e)来表示指数部分，如123.456或123e-2。
4. int atoi(const char *s)
   int atol(const char *s)
        字符串转换成整数,貌似在发生溢出时，并不设置errno。
        在发生溢出时貌似只是简单处理，从而不可靠，难以判断，在linux上待验证
5. strtod(const char *s, char **endp)
        strtod()会扫描参数s字符串，跳过前面的空格字符，直到遇上数字或正负符号才开始做转换，到出现非数字或字符串结束时('\0')才结束转换，并将结果返回。
        若endp不为NULL，则会将遇到不合条件而终止的nptr中的字符指针由endp传回。参数s字符串可包含正负号、小数点或E(e)来表示指数部分。如123.456或123e-2。
6. strtol(const char *s, char **endp, int base)
        字符串转换成整数，base表示以多少进制解析，比如2，8，10，16，取0则默认以10进制解析，但如果s有前导0x则以16进制解析，有0以8进制解析。
        在发生上溢或下溢时，会将返回结果置为LONG_MAX或LONG_MIN (limits.h),并设置errno为ERANGE (errno.h)，由此可以判断是否溢出。
        在转换过程中，若有溢出的风险或需判断溢出，则最好使用strtol函数。
7. strtoul(const char *s, char **endp, int base)
        在strtol类似，只是转换为unsigned long类型。溢出时返回ULONG_MAX(limits.h)
